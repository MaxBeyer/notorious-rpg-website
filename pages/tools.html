Get a Mutation

Reactive Effects

Some mutations have what’s called reactive effects (R). In a conflict, you can activate a reactive effect any time you want, and any number of times in a single turn. Activating a reactive effect does not count as an action or as a maneuver – all you need is MP to spend.

Enhancing Effects

Another type of mutation effect is the enhancing effect (E). You activate an enhancing effect at the same time as you perform an action or a maneuver, to enhance it. Using an enhancing effect does not count as an action or a maneuver in itself. When using an enhancing effect with a skill roll, roll for the mutation first and then for the skill.

Generate a Sector

The first thing we need to do when generating a sector is to determine the Threat Level. This will help determine how many threats and artifacts are present in the sector.

If you feel like being nice to your players, try something low (1-4).

If you think they are up for a challenge, crank it up a few ticks (5-8).

If you're an absolute monster and want to see your players perish, go even higher (9+).
